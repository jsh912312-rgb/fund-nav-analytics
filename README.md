## 성과지표 설계 및 계산 방법

### 1. 누적수익률 (Cumulative Return)

**개념:** 투자 시작부터 현재까지 총 몇 % 수익을 냈는가

**계산 원리:** 일별 수익률을 단순 합산이 아닌 복리로 곱해 계산한다

```
누적수익률 = (1 + r₁) × (1 + r₂) × ... × (1 + rₙ) - 1
```

**코드:**

```python
daily_ret = group['daily_return'].dropna()
# pct_change()로 생성한 일별 수익률에 1을 더해 전부 곱한 뒤 1을 뺌
cumulative_return = (1 + daily_ret).prod() - 1
```

- `pct_change()`: 전일 대비 수익률 → `(오늘 NAV - 어제 NAV) / 어제 NAV`
- `.prod()`: 전체 원소를 모두 곱함 (복리 누적)
- `dropna()`: 첫날은 전일 데이터가 없어 NaN이므로 제거

### 2. CAGR (연환산 수익률, Compound Annual Growth Rate)

**개념:** 누적수익률을 1년 기준으로 평탄화한 수치. 기간이 다른 펀드를 비교할 때 사용한다

**계산 원리:**

```
CAGR = (1 + 누적수익률)^(1/투자연수) - 1
```

**코드:**

```python
n_years = len(daily_ret) / 252   # 거래일 기준 1년 = 252일
cagr = (1 + cumulative_return) ** (1 / n_years) - 1
```

- `252`: 주식시장 연간 거래일 수 (365일 중 주말·공휴일 제외)
- `* (1/n_years)`: n년치 복리를 1년으로 역산하는 거듭제곱

### 3. 변동성 (Volatility)

**개념:** 일별 수익률의 표준편차를 연환산한 값. 높을수록 가격 변동이 크고 투자 리스크가 높다

**계산 원리:**

```
연환산 변동성 = 일별 표준편차 × √252
```

√252를 곱하는 이유: 수익률의 분산이 시간에 비례한다는 통계적 가정

```
연간 분산 = 일별 분산 × 252
연간 표준편차 = 일별 표준편차 × √252 (≈ 15.87)
```

**코드:**

```python
volatility = daily_ret.std() * np.sqrt(252)
```

- `.std()`: 일별 수익률의 표준편차
- `np.sqrt(252)`: 연환산 계수 ≈ 15.87

### 4. 샤프지수 (Sharpe Ratio)

**개념:** 위험 1단위당 초과수익. 높을수록 같은 위험으로 더 많이 번 것이다

**계산 원리:**

```
샤프지수 = (CAGR - 무위험수익률) / 변동성
```

- 분자: 무위험자산(국고채 3.5%) 대비 얼마나 더 벌었는가
- 분모: 그 수익을 얻기 위해 감수한 위험(변동성)

해석 기준:
- 1.0 이상: 우수
- 0 ~ 1.0: 보통
- 0 미만: 무위험자산(예금)보다 비효율

**코드:**

```python
rf = 0.035   # 국고채 3년물 기준 무위험수익률
sharpe = (cagr - rf) / volatility if volatility != 0 else 0
```

### 5. MDD (최대낙폭, Maximum DrawDown)

**개념:** 투자 기간 중 최고점 대비 가장 많이 떨어진 순간의 손실폭. 0에 가까울수록 방어적인 펀드다

**계산 원리:**

```
Drawdown(t) = (현재 NAV - 그때까지의 최고 NAV) / 그때까지의 최고 NAV
MDD = Drawdown 중 최솟값
```

**코드:**

```python
nav_series = group['nav']
rolling_max = nav_series.cummax()                         # 누적 최고점
drawdown = (nav_series - rolling_max) / rolling_max      # 각 시점 낙폭
mdd = drawdown.min()                                      # 최대 낙폭
```

- `cummax()`: 해당 시점까지의 누적 최댓값 (고점 갱신 시 업데이트)
- `.min()`: 낙폭 시리즈에서 가장 작은(가장 많이 떨어진) 값

### 6. 초과수익 (Excess Return)

**개념:** 같은 기간 KOSPI 수익률과 비교해 펀드가 얼마나 더/덜 벌었는가

**계산 원리:**

```
초과수익 = 펀드 누적수익률 - KOSPI 누적수익률
```

KOSPI 수익률은 동일 기간 데이터만 사용해야 하므로 `inner join`으로 날짜를 맞춘다.

**코드:**

```python
merged = group.merge(kospi_df, on='date', how='inner')   # 날짜 기준 병합
kospi_return = (merged['kospi'].iloc[-1] / merged['kospi'].iloc[0]) - 1
excess_return = cumulative_return - kospi_return
```

- `how='inner'`: 양쪽에 모두 존재하는 날짜만 사용 (공휴일 불일치 방지)
- `iloc[-1] / iloc[0] - 1`: 마지막값 / 첫값 - 1 = 기간 누적수익률

## 인사이트 분석

1. 핵심 발견: 수익률 1위 ≠ 최고의 투자
    
    KODEX레버리지는 누적수익률 **+590%** 로 압도적 1위지만, 샤프지수는 0.85로 6위다. 반면 TIGER미국S&P500은 누적수익률 +192%로 5위이지만 샤프지수는 1.23으로 1위다. 단순 수익률이 아닌 리스크 조정 성과 기준으로 분석해야 투자자에게 의미 있는 정보를 줄 수 있다.
    
2. 미국 시장 ETF의 우위
    
    TIGER미국S&P500은 변동성 **15.62%** 로 가장 낮고, MDD는 **-19.15%** 로 가장 방어적이며, 샤프지수는 **1.23** 으로 가장 높다. 분석 기간(2021~2026) 동안 미국 시장의 구조적 강세가 국내 ETF 대비 월등한 리스크 조정 성과로 이어졌다.
    
3. KOSPI 대비 초과수익
    
    국내 시장 추종 ETF(TIGER200, KODEX200)는 KOSPI 대비 각각 +104%, +102%의 초과수익을 냈다. 이는 배당 재투자 효과와 일부 추적 오차의 영향으로 분석된다. 반면 KODEX코스닥150(-106%), TIGER차이나(-130%), KODEX인버스(-212%)는 KOSPI 대비 크게 부진했다.

## 시계열 분석 및 예측

### Prophet 모델 개요

Facebook(Meta)이 개발한 시계열 예측 라이브러리로, 트렌드·계절성·이벤트(휴일 효과)를 분리해 모델링한다. 주가처럼 추세가 있고 계절성이 뚜렷한 시계열 데이터에 적합하다.

`NAV = 트렌드(trend) + 주간 계절성(weekly) + 연간 계절성(yearly) + 오차(ε)`

### Prophet 예측 코드

샤프지수 1위인 TIGER미국S&P500을 대상으로 향후 60 영업일 NAV를 예측한다.

**주요 파라미터 설명**

| 파라미터 | 설정값 | 의미 |
| --- | --- | --- |
| `changepoint_prior_scale` | 0.05 | 추세 변화점 민감도. 낮으면 완만, 높으면 급격한 추세 변화 반영 |
| `weekly_seasonality` | True | 요일별 패턴 반영 (주식은 월~금만 거래) |
| `yearly_seasonality` | True | 연간 주기 패턴 반영 (연말 랠리 등) |
| `freq='B'` | 영업일 | 주말·공휴일 제외한 거래일 기준 예측 |

### Prophet 예측 시각화

**Prophet 예측 결과 해석**

TIGER미국S&P500은 2021년 약 10,000에서 현재 약 25,000 수준까지 꾸준히 상승했다. Prophet 모델은 이 상승 추세를 학습해 향후 60 영업일도 완만한 우상향을 예측한다. 다만 예측 구간이 넓어질수록 신뢰구간이 벌어지는 것은 불확실성이 시간이 지날수록 커지기 때문이다.

**한계:** 금리 변화, 환율, 지정학 리스크 등 외부 변수를 반영하지 않아 참고용으로만 활용해야 한다.

### 시장 국면별 성과 비교

동일 기간 10개 ETF의 수익률을 겹쳐서 보여줌으로써, 시장 환경에 따라 어떤 ETF가 강세/약세를 보이는지 확인한다. 각 펀드의 기간 시작 시점 NAV를 기준(0%)으로 정규화해 비교한다.

**정규화 핵심 코드:**

```python
nav_normalized = (group['nav'] / group['nav'].iloc[0] - 1) * 100
# 예: 시작 NAV = 10,000, 현재 NAV = 11,500 → (11500/10000 - 1) × 100 = +15%
```

## 한계점 및 향후 계획

1. 현재 한계
    - 수수료·세금 미반영으로 실제 투자 수익과 차이 발생 가능
    - Prophet 예측은 외부 변수(금리·환율·지정학 리스크) 미반영으로 참고용 수준
    - 무위험수익률을 3.5% 고정값으로 가정 (실제로는 시기별로 변동)
    - ETF 10종만 분석하여 전체 공모펀드 시장을 대표하기 어려움
2. 향후 개선 계획
    - Factor 모델(Fama-French 3·5 Factor) 적용으로 초과수익 원인 분해
    - PostgreSQL 마이그레이션으로 대용량 데이터 처리 성능 개선
    - 무위험수익률 동적 반영 (한국은행 금리 API 연동)
    - Rolling Sharpe Ratio 시각화로 시장 국면별 성과 변화 추적
    - 실시간 알림 기능 추가 (MDD 임계값 초과 시 알림)
